
# News Monkey

 A simple news website build in react js using newsapi.org

### Instalation

 - git clone git@github.com:devocoe/News-Monkey.git
 - cd News-Monkey
 - yarn install 
 - create .env file in root directory
 - Add in env file REACT_APP_API_KEY=`ENTER_HERE_YOUR_API_KEY`
 - yarn start
